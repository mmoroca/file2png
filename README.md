## Universal File â†” PNG Converter (via ZIP)

Bi-directional Python tool updated to convert **any file type** into PNG images (bitmaps) and vice versa. Based on the visual design by MrTalida (https://x.com/MrTalida), this version adds an automatic **ZIP** compression layer to maximize efficiency and allow storing multiple files or complex formats within a single image.

### ðŸ“Œ Features

* **Universal Support:** Converts any extension (`.pdf`, `.exe`, `.txt`, `.jpg`) into a `.png` image.
* **Smart Compression:**
    * If the input file is standard, it is compressed into a **ZIP** in memory before generating the image.
    * If the input file is already a `.zip`, its binary data is processed directly to avoid double compression.
* **Gray-Tone Agnostic:** When reading a PNG, it ignores gray padding and processes only Black/White pixels.
* **Square Format:** The output image will always be square, padding any remaining pixels with **Gray (127)**.

### ðŸš€ Usage

Ensure you have Python 3 and the **Pillow** library installed:
1.  **Installation:** `pip install Pillow`

The script is run from the command line. The tool automatically detects whether you are encoding or decoding based on the file extension.

`python file2png.py <source> <destination>`

#### Parameters

| Parameter | Description |
| :--- | :--- |
| `<source>` | **To create image:** Any file (`.pdf`, `.docx`, `.zip`).<br>**To extract:** A `.png` image generated by this script. |
| `<destination>` | **To create image:** Output filename (`.png`).<br>**To extract:** Path to the **folder** where recovered files will be saved. |

### Usage Examples

1.  **Encode a document (Any File â†’ PNG)**
    *Compresses `contract.pdf`, converts it to bits, and saves the image.*
    `python file2png.py contract.pdf secure_image.png`

2.  **Encode an existing ZIP (ZIP â†’ PNG)**

3.  **Decode (PNG â†’ Folder)**
    *Reads the image, extracts the internal ZIP, and saves content to `recovered_files` folder.*
    `python file2png.py secure_image.png ./recovered_files`
